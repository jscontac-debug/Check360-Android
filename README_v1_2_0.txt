Check 360 Covirán - v1.2.0

Cambios principales:
- Nueva imagen visual moderna.
- Barra de progreso por sección.
- Tarjetas más limpias para preguntas, fotos y estado de la visita.
- Mantiene cámara nativa, compresión de fotos y estructura de actualización.

Para generar APK:
1. npm install
2. npx cap sync android
3. cd android
4. gradlew assembleDebug

APK resultante:
android\app\build\outputs\apk\debug\app-debug.apk
