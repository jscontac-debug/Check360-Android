Check360 Android v1.4.0

Objetivo: rediseño sobrio y corporativo.

Cambios principales:
- Uso moderado del verde Covirán aproximado RAL 6037.
- Eliminados degradados y aspecto excesivamente colorido.
- Pantalla inicial más limpia y profesional.
- Selector preparado para futuras plantillas Check360 / Check180.
- Botones, tarjetas y campos con estética más empresarial.
- Versión interna actualizada a 1.4.0.

Compilación:
npm install
npx cap sync android
cd android
gradlew assembleDebug

APK:
android/app/build/outputs/apk/debug/app-debug.apk
